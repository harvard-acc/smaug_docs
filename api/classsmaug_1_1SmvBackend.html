class smaug::SmvBackend¶

SmvBackend implements a set of models of optimized DL kernels that were taped out on a machine learning SoC by the Harvard Architecture, Circuits, and Compilers. All operators implemented in this backend are vectorized using eight fp16 values. See the individual operators for more details.
