13 // Matrix b after transposition:
14 //
15 // cols (originally rows) --->
16 //
17 // rows [[---][---][---]]
18 // | [[---][---][---]]
19 // [[---][---][---]]
20 // v [[---][---][---]]
21 //
22 // Each [---] represents an 8-wide vector. This inner product executes a 32-way
23 // MACC -- 4 such 8-wide vectors -- per PE, and 8 PEs, where each PE is
24 // assigned a row in in the transposed matrix. It continues across each row of
25 // b until the complete output pixel is finished (output stationary).
26 //
27 // No biases are added.
28 //
29 // Args:
30 // host_a: Host buffer for a in NC.
31 // host_b: Host buffer for b in NC.
32 // host_results: Host results buffer in NC.
33 // a: Local buffer for a in NC.
34 // b: Local buffer for b in NC.
35 // results: Local results buffer in NC.
36 // a_dims: Dimensions of a.
37 // b_dims: Dimensions of b.
38 // results_dims: Dimensions of the results.
39 // a_pad: Align padding size on the channel dimension of a.
40 // b_pad: Align padding size on the channel dimension of b.
41 // results_pad: Align padding size on the channel dimension of the results.
42 // a_start: If a contains more activations than b, start from this one.
43 // Otherwise this should always be zero.
44 // result_start: If the results contain more neurons than the b, start writing
45 // results from this one. Otherwise this should always be zero.
46 // accumulate: If the original b tensor is tiled on activations, this should
47 // be set to true in order to avoid resetting the result buffer for
48 // non-first b tiles.
49 // send_results: Send the results to the host memory if this is true.
50 // act_function: Activation function the operator runs.
51 // act_params: Parameters for the activation function.
